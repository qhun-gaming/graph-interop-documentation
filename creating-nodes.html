Creating Custom Nodes

This guide explains how to create custom nodes for use in GraphInterop graphs.

Basic Node Structure

Every node must:

  1. Extend GraphNode

  2. Add [NodeInfo] attribute for metadata

  3. Mark fields with [Input] or [Output]

  4. Implement Process()

using QhunGaming.GraphInterop; [NodeInfo(Category = "Math", Title = "Add")] public class AddNode : GraphNode { [Input(Required = true)] public float a; [Input(Required = true)] public float b; [Output] public float result; public override void Process(GraphContext context) { result = a + b; } }

Attributes

NodeInfo Attribute

Provides metadata for the node search window:

[NodeInfo( Category = "Math/Trigonometry", // Hierarchical path Title = "Sine", // Display name Description = "Calculates sine" // Tooltip )]

Input Attribute

Marks a field as an input port:

[Input( DisplayName = "Value", // UI label (default: field name) Required = true, // Must be connected or have value ShowEmbedded = true, // Show inline editor Description = "Input value" // Tooltip )] public float value;

Output Attribute

Marks a field as an output port:

[Output( DisplayName = "Result", Description = "Computed result" )] public float result;

Common Patterns

Constant Value Node

[NodeInfo(Category = "Constants", Title = "Float")] public class FloatConstantNode : GraphNode { [Input(Required = false, ShowEmbedded = true)] [SerializeField] private float _value = 0f; [Output] public float value; public override void Process(GraphContext context) { value = _value; } }

Pass-through Node

[NodeInfo(Category = "Flow", Title = "Branch")] public class BranchNode : GraphNode { [Input(Required = true, ShowEmbedded = true)] public bool condition; [Input(Required = true)] public object trueValue; [Input(Required = true)] public object falseValue; [Output] public object result; public override void Process(GraphContext context) { result = condition ? trueValue : falseValue; } }

Collection Processing

[NodeInfo(Category = "Collections", Title = "Filter")] public class FilterNode : GraphNode { [Input(Required = true)] public List<float> input; [Input(ShowEmbedded = true)] public float threshold; [Output] public List<float> output; public override void Process(GraphContext context) { output = input.Where(x => x > threshold).ToList(); } }

Using Context

[NodeInfo(Category = "Input", Title = "Get Spline")] public class SplineInputNode : GraphNode { [Output] public ISplineContainer spline; public override void Process(GraphContext context) { spline = context.Get<ISplineContainer>("spline"); } }

With Validation

[NodeInfo(Category = "Math", Title = "Divide")] public class DivideNode : GraphNode { [Input] public float dividend; [Input] public float divisor; [Output] public float result; public override void Process(GraphContext context) { if (divisor == 0) throw new DivideByZeroException(); result = dividend / divisor; } public override void Validate(List<string> errors) { // Check embedded value if not connected if (divisor == 0) errors.Add("Divisor cannot be zero"); } }

Type Guidelines

Supported Types

Category

Types

Primitives

int, float, bool, string

Unity Types

Vector3, Color, AnimationCurve

Collections

List<T>, T[]

Custom

Any [Serializable] class/struct

Interfaces

For polymorphism

Flexible Typing

Use object for maximum flexibility:

[Input] public object anyInput; // Accepts any type [Output] public object anyOutput; // Can output any type

Interface Types

[Input] public IEnumerable<float> data; // Accepts List<float>, float[], etc.

Best Practices

Do

  • Keep nodes focused (single responsibility)

  • Use descriptive names and descriptions

  • Set Required = false for optional inputs

  • Use ShowEmbedded = true for common defaults

  • Validate inputs in Validate() method

Don't

  • Access Unity APIs that require main thread in Process()

  • Store state between executions (nodes should be stateless)

  • Use static fields for data sharing (use context)

  • Create complex inheritance hierarchies

Debug Output

[NodeInfo(Category = "Debug", Title = "Log")] public class DebugLogNode : GraphNode { [Input(Required = true)] public object value; [Input(ShowEmbedded = true)] public string prefix = ""; public override void Process(GraphContext context) { Debug.Log($"{prefix}{value}"); } }
