GraphInterop Help

Editor API

The editor layer provides visual graph editing capabilities through two backend adapters: GraphView (Unity 2019+) and GraphToolkit (Unity 6.2+).

Assembly

Name: QhunGaming.GraphInterop.Editor Namespace: QhunGaming.GraphInterop.Editor

Architecture

Backend Selection

GraphView (Default)

Available in all Unity versions with UIElements support.

Compilation: Always compiled when UNITY_EDITOR is defined.

GraphToolkit

Experimental API available in Unity 6.2+.

Compilation: Only when both USING_GTK and UNITY_6000_2_OR_NEWER are defined.

Common Features

Both backends provide:

  • Visual node editing with drag & drop

  • Port connection with type compatibility checking

  • Node search/creation window

  • Blackboard for shared variables

  • Zoom and pan navigation

  • Selection and multi-select

  • Undo/redo support

  • Copy/paste (planned)

Creating an Editor Window

Subclass GraphInteropEditorWindow

public class MyPipelineEditor : GraphInteropEditorWindow { [MenuItem("Window/My Pipeline Editor")] public static void Open() { GetWindow<MyPipelineEditor>("My Pipeline"); } // Optionally filter which node types appear protected override bool CanCreateNodeType(Type nodeType) { return nodeType.Namespace?.Contains("MyNamespace") ?? false; } }

Custom Asset Handling

public class MyPipelineEditor : GraphInteropEditorWindow { protected override void OnGraphLoaded(GraphAsset asset) { base.OnGraphLoaded(asset); titleContent.text = asset.name; } protected override void OnGraphChanged(GraphAsset asset) { base.OnGraphChanged(asset); // Custom change handling } }

Port Styling

Ports can be styled based on type in USS:

/* By type attribute */ .port[data-type="System.Single"] .connector { background-color: #7fc97f; } .port[data-type="UnityEngine.Vector3"] .connector { background-color: #fdc086; }

Node Search Window

The search window automatically discovers nodes marked with [NodeInfo]:

[NodeInfo(Category = "Math/Basic", Title = "Add")] public class AddNode : GraphNode { }

Creates the hierarchy:

Math └── Basic └── Add
