GraphAsset

Abstract ScriptableObject that contains all graph data: nodes, connections, and blackboard variables.

Class Definition

namespace QhunGaming.GraphInterop { public abstract class GraphAsset : ScriptableObject { public IReadOnlyList<GraphNode> Nodes { get; } public IReadOnlyList<Connection> Connections { get; } public Blackboard Blackboard { get; } public void AddNode(GraphNode node); public void RemoveNode(GraphNode node); public void RemoveNode(string nodeId); public GraphNode GetNode(string id); public void AddConnection(Connection connection); public void AddConnection(string sourceNodeId, string sourcePortName, string targetNodeId, string targetPortName); public void RemoveConnection(Connection connection); public Connection GetConnectionToInput(string nodeId, string portName); public IEnumerable<Connection> GetConnectionsFromOutput(string nodeId, string portName); public void Clear(); public List<string> Validate(); } }

Properties

Nodes

  • Type: IReadOnlyList<GraphNode>

  • Description: All nodes in the graph

  • Serialization: Uses [SerializeReference] for polymorphism

Connections

  • Type: IReadOnlyList<Connection>

  • Description: All connections between ports

Blackboard

  • Type: Blackboard

  • Description: Container for shared variables

Node Management

AddNode(GraphNode)

Adds a node to the graph.

var node = new AddNode(); graphAsset.AddNode(node); // node.Id is auto-generated if empty

RemoveNode(GraphNode) / RemoveNode(string)

Removes a node and all its connections.

graphAsset.RemoveNode(node); // or graphAsset.RemoveNode(nodeId);

GetNode(string)

Retrieves a node by ID.

var node = graphAsset.GetNode("abc-123");

Connection Management

AddConnection(Connection)

Adds a connection between two ports. Automatically removes any existing connection to the same input port (inputs can only have one connection).

graphAsset.AddConnection(new Connection( sourceNodeId: "node1", sourcePortName: "output", targetNodeId: "node2", targetPortName: "input" ));

Overloaded Version

graphAsset.AddConnection("node1", "output", "node2", "input");

GetConnectionToInput(string, string)

Gets the single connection to a specific input port.

var conn = graphAsset.GetConnectionToInput(nodeId, "inputName"); if (conn != null) { // Port is connected }

GetConnectionsFromOutput(string, string)

Gets all connections from an output port (outputs can have multiple connections).

foreach (var conn in graphAsset.GetConnectionsFromOutput(nodeId, "outputName")) { // Handle each connection }

Validation

Validate()

Validates the entire graph and returns errors.

var errors = graphAsset.Validate(); if (errors.Count > 0) { foreach (var error in errors) Debug.LogError(error); }

Validation checks:

  • Required inputs have connections or embedded values

  • Graph contains no cycles

  • Custom node validation via GraphNode.Validate()

Usage Example

// Create and populate a graph var graph = ScriptableObject.CreateInstance<MyGraphAsset>(); var inputNode = new SplineInputNode(); var processNode = new SpacingNode(); var outputNode = new DebugOutputNode(); graph.AddNode(inputNode); graph.AddNode(processNode); graph.AddNode(outputNode); graph.AddConnection(inputNode.Id, "spline", processNode.Id, "input"); graph.AddConnection(processNode.Id, "output", outputNode.Id, "data"); // Validate var errors = graph.Validate(); if (errors.Count == 0) { // Execute var executor = new GraphExecutor(); executor.Execute(graph, new GraphContext()); }
