InteropGraphView

Main container that manages all visual elements and synchronizes with GraphAsset.

Key Methods

public class InteropGraphView : GraphView { // Load a graph asset into the view public void LoadGraph(GraphAsset asset); // Add a node at a position public void AddNode(GraphNode node, Vector2 position); // Create node view for an existing graph node public InteropNodeView CreateNodeView(GraphNode node); // Event when graph is modified public event Action<GraphAsset> OnGraphChanged; }

Port Compatibility

Override of GetCompatiblePorts implements type-aware filtering:

public override List<Port> GetCompatiblePorts(Port startPort, NodeAdapter nodeAdapter) { // Uses TypeCompatibility.AreCompatible() to filter valid connections }

Built-in Manipulators

  • ContentDragger - Pan the view

  • SelectionDragger - Move selected elements

  • RectangleSelector - Box selection

  • ContentZoomer - Zoom in/out

InteropNodeView

Visual representation of a GraphNode.

Features

  • Title from [NodeInfo] attribute or type name

  • Input/output ports generated from attributes

  • Embedded value editors for unconnected inputs

  • Position persistence

Port Management

public class InteropNodeView : Node { public GraphNode Node { get; } public GraphAsset GraphAsset { get; set; } // Get port by name public InteropPort GetInputPort(string name); public InteropPort GetOutputPort(string name); // Update variable node display public void UpdateVariableDisplay(); }

InteropPort

Custom port with additional type information.

public class InteropPort : Port { public string PortName { get; } public Type PortDataType { get; } public PortDefinition Definition { get; } public static InteropPort Create( PortDefinition definition, Orientation orientation, Direction direction, Port.Capacity capacity); }

Port Creation

// Single connection input var input = InteropPort.Create( portDef, Orientation.Horizontal, Direction.Input, Port.Capacity.Single); // Multi connection output var output = InteropPort.Create( portDef, Orientation.Horizontal, Direction.Output, Port.Capacity.Multi);

InteropBlackboard

Displays and manages shared variables.

Features

  • Add new variables via button

  • Drag variables to create GetVariable nodes

  • Edit variable names and types

  • Delete variables

public class InteropBlackboard : Blackboard { public void LoadVariables(GraphAsset asset); public event Action<GraphAsset> OnVariableChanged; }

NodeSearchWindow

Searchable menu for creating nodes.

Node Discovery

Automatically finds all types with [NodeInfo]:

[NodeInfo(Category = "Math/Trigonometry", Title = "Sin")] public class SinNode : GraphNode { }
