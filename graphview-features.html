GraphView Features in GraphInterop

This page explains how GraphInterop uses Unity's GraphView for visual graph editing and what features are supported.

Overview

GraphView is Unity's UI Toolkit-based graph visualization framework. While considered legacy (GraphToolkit is the intended replacement), GraphView provides a mature, feature-rich editor experience.

Why Use GraphView?

  1. Mature and Stable: Battle-tested in Unity's official tools (Shader Graph, Visual Effect Graph)

  2. Feature-Complete UI: Robust visual editing capabilities

  3. Production-Ready: Not experimental (unlike GraphToolkit UI)

  4. UI Toolkit Integration: Modern Unity UI framework

GraphInterop's Approach

GraphInterop uses GraphView for the editor experience only:

  • Runtime API is GraphToolkit-based

  • Editor UI is GraphView-based

  • Clean separation via IGraphEditorBackend

GraphView Backend Implementation

GraphInterop uses a backend abstraction to support multiple editors:

public interface IGraphEditorBackend { GraphAsset CurrentGraph { get; } void Initialize(EditorWindow window, VisualElement root); void LoadGraph(GraphAsset graph); void OnDisable(); }

GraphView Backend:

public class GraphViewEditorBackend : IGraphEditorBackend { private InteropGraphView _graphView; private InteropBlackboard _blackboard; private InteropInspector _inspector; public void Initialize(EditorWindow window, VisualElement root) { // Create GraphView _graphView = new InteropGraphView(window); // Create Blackboard _blackboard = new InteropBlackboard(_graphView); // Create Inspector _inspector = new InteropInspector(); // Layout root.Add(_graphView); _graphView.Add(_blackboard); _graphView.Add(_inspector); } }

Registration:

[InitializeOnLoadMethod] private static void RegisterBackend() { GraphInteropEditorWindow.RegisterBackend(() => new GraphViewEditorBackend()); }

This allows future GraphToolkit backend without changing user code.

GraphView Limitations and Workarounds

Limitation 1: No Built-in Minimap Zoom

Issue: GraphView minimap doesn't zoom properly.

Workaround:

// Custom minimap scaling minimap.SetZoomScale(graphView.scale);

Limitation 2: Port Reordering

Issue: Changing port order requires node rebuild.

Workaround:

// Define ports in desired order protected override void OnDefinePorts(IPortDefinitionContext context) { // Inputs first context.AddInputPort<float>("input1").Build(); context.AddInputPort<float>("input2").Build(); // Outputs last context.AddOutputPort<float>("output").Build(); }

Limitation 3: Embedded Field Types

Issue: GraphView supports limited embedded field types (primitives, Vector3, Color, etc.).

Workaround:

// Use custom property drawers [CustomPropertyDrawer(typeof(MyCustomType))] public class MyCustomTypeDrawer : PropertyDrawer { } // Or use multiple ports context.AddInputPort<float>("x").Build(); context.AddInputPort<float>("y").Build(); context.AddInputPort<float>("z").Build();

GraphView Integration Details

Node View Creation

public class InteropNodeView : Node { public GraphNode NodeModel { get; } public InteropNodeView(GraphNode model) { NodeModel = model; // Set title title = model.GetType().GetCustomAttribute<NodeInfoAttribute>()?.DisplayName ?? model.GetType().Name; // Create ports foreach (var port in model.GetInputPorts()) { var portView = CreatePort(port); inputContainer.Add(portView); } foreach (var port in model.GetOutputPorts()) { var portView = CreatePort(port); outputContainer.Add(portView); } // Create option fields foreach (var option in model.nodeOptions) { var field = CreateOptionField(option); extensionContainer.Add(field); } } }

Port View Creation

public class InteropPort : Port { public IPort PortModel { get; } public InteropPort(IPort model, Direction direction, Capacity capacity) : base(Orientation.Horizontal, direction, capacity, model.dataType) { PortModel = model; portName = model.displayName; portColor = GetPortColor(model.dataType); // Embedded value field if (direction == Direction.Input && !model.isConnected) { ShowEmbeddedField(); } } private void ShowEmbeddedField() { var field = CreateFieldForType(PortModel.dataType); field.value = PortModel.EmbeddedValue; field.RegisterValueChangedCallback(e => { PortModel.EmbeddedValue = e.newValue; }); Add(field); } }

Edge Creation

public void OnDrop(GraphView graphView, Edge edge) { var outputPort = edge.output as InteropPort; var inputPort = edge.input as InteropPort; var outputNode = outputPort.node.userData as GraphNode; var inputNode = inputPort.node.userData as GraphNode; // Create connection in model var connection = new Connection { SourceNodeId = outputNode.Id, SourcePortName = outputPort.PortModel.name, TargetNodeId = inputNode.Id, TargetPortName = inputPort.PortModel.name }; graphAsset.Connections.Add(connection); // Create edge in view var edgeView = new Edge { output = outputPort, input = inputPort }; graphView.AddElement(edgeView); }

// Global port color rules public static class PortColorRegistry { private static Dictionary<Type, Color> _colors = new() { { typeof(float), new Color(0.5f, 0.8f, 1.0f) }, // Light blue { typeof(int), new Color(0.5f, 1.0f, 0.5f) }, // Light green { typeof(string), new Color(1.0f, 0.8f, 0.5f) }, // Light orange { typeof(Transform), new Color(1.0f, 0.5f, 0.5f) } // Light red }; public static Color GetColor(Type type) { if (_colors.TryGetValue(type, out var color)) return color; // Default color return new Color(0.8f, 0.8f, 0.8f); } }
