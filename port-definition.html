GraphInterop Help

PortDefinition

Represents metadata about a port on a node, including its name, type, direction, and configuration.

Class Definition

namespace QhunGaming.GraphInterop { public enum PortDirection { Input, Output } public class PortDefinition { public string Name { get; } public string DisplayName { get; } public Type DataType { get; } public PortDirection Direction { get; } public bool Required { get; } public bool ShowEmbedded { get; } public string Description { get; } public FieldInfo FieldInfo { get; } public static PortDefinition FromInput(FieldInfo field, InputAttribute attr); public static PortDefinition FromOutput(FieldInfo field, OutputAttribute attr); public object GetValue(GraphNode node); public void SetValue(GraphNode node, object value); } }

Properties

Name

  • Type: string

  • Description: The field name used as port identifier

  • Example: "inputValue", "result"

DisplayName

  • Type: string

  • Description: Human-readable name shown in UI

  • Example: "Input Value", "Result"

  • Auto-formatting: CamelCase to Title Case

DataType

  • Type: Type

  • Description: The CLR type of data this port handles

  • Example: typeof(float), typeof(List<Vector3>)

Direction

  • Type: PortDirection

  • Values: Input, Output

Required

  • Type: bool

  • Description: Whether this input must be connected or have a value

  • Applies to: Input ports only

ShowEmbedded

  • Type: bool

  • Description: Whether to show inline value editor in node UI

  • Applies to: Input ports only

Description

  • Type: string

  • Description: Tooltip text for the port

FieldInfo

  • Type: System.Reflection.FieldInfo

  • Description: Reflection info for accessing the field

Factory Methods

FromInput(FieldInfo, InputAttribute)

Creates a port definition from a field with [Input] attribute.

var portDef = PortDefinition.FromInput(fieldInfo, inputAttribute);

FromOutput(FieldInfo, OutputAttribute)

Creates a port definition from a field with [Output] attribute.

var portDef = PortDefinition.FromOutput(fieldInfo, outputAttribute);

Value Access

GetValue(GraphNode)

Gets the current value of the port from a node instance.

var value = portDef.GetValue(node);

SetValue(GraphNode, object)

Sets the value of the port on a node instance.

portDef.SetValue(node, 42.0f);

Usage in Editor

Example: Custom Node with Ports

public class TransformNode : GraphNode { [Input( DisplayName = "Position", Required = true, ShowEmbedded = false, Description = "World position to transform" )] public Vector3 position; [Input( DisplayName = "Scale", Required = false, ShowEmbedded = true )] [SerializeField] private float scale = 1.0f; [Output( DisplayName = "Result", Description = "Transformed position" )] public Vector3 result; public override void Process(GraphContext context) { result = position * scale; } }

Internal Implementation

Port definitions are created via reflection and cached:

// In GraphNode public IReadOnlyList<PortDefinition> GetInputPorts() { if (_inputPorts == null) { _inputPorts = GetType() .GetFields(BindingFlags.Public | BindingFlags.NonPublic | BindingFlags.Instance) .Select(f => new { Field = f, Attr = f.GetCustomAttribute<InputAttribute>() }) .Where(x => x.Attr != null) .Select(x => PortDefinition.FromInput(x.Field, x.Attr)) .ToList(); } return _inputPorts; }
